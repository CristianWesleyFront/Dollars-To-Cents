## 📓 Sobre

Referência do projeto [app-ideas](https://github.com/florinpop17/app-ideas) de [florinpop17](https://github.com/florinpop17)

Essa pequena aplicação permite inserir uma certa quantidade de dólares e retorna sua quantidade em moedas americanas: Half dollar = 50 centavos, quarter dollar = 25 centavos, dime = 10 centavos, nickel = 5 centavos , penny = 1 centavo.

### Ações de usuários

- [x] ~O usuário pode inserir um valor em dólar~
- [x] ~O usuário pode ver o total de centavos do valor do dólar convertido~

## 📦 Como baixar o projeto

### Com [Yarn](https://yarnpkg.com/) :

```bash

 # Clonar o repositório
 $ git clone https://github.com/CristianWesleyFront/Dollars-To-Cents

 # Entrar no diretorio
 $ cd Dollars-To-Cents

 # Instalar as dependências
 $ yarn install

 # Iniciar o projeto
 $ yarn start


```

### Com [Npm](https://www.npmjs.com/) :

```bash

 # Clonar o repositório
 $ git clone https://github.com/CristianWesleyFront/Dollars-To-Cents

 # Entrar no diretorio
 $ cd Dollars-To-Cents

 # Instalar as dependências
 $ npm i

 # Iniciar o projeto
 $ npm run start


```
